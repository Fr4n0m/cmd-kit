# Code Of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of acceptable behavior:

- giving actionable technical feedback
- assuming good intent while discussing tradeoffs
- reporting bugs and security issues responsibly
- keeping criticism focused on code, behavior, or process

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- discriminatory language or exclusionary behavior
- publishing private information without consent
- derailing technical discussions with hostility or bad-faith argument

## Enforcement

Project maintainers may edit, remove, or reject contributions and interactions that do not meet this standard.

## Reporting

If you need to report conduct issues, contact the maintainers privately through the security contact process in [SECURITY.md](./SECURITY.md) until a dedicated public channel is defined.
