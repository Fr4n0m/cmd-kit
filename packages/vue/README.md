# @cmd-kit/vue

Vue bindings for `cmd+kit`.

This package provides a Vue composable and default command palette component on top of `@cmd-kit/core`.

## Scope

- `CommandPalette` component
- `useCommandPalette` composable
- grouped command rendering
- keyboard navigation
- nested navigation
- async command sources
- recent item tracking
- theming through props
- slot-based customization for title, sections, items, and empty state
