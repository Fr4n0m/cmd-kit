# @cmd-kit/react

React bindings for `cmd+kit`.

This package provides the default React command palette UI on top of `@cmd-kit/core`.

## Scope

- `CommandPalette` component
- grouped command rendering
- keyboard navigation
- nested navigation
- theming through props
- theme tokens that can also be converted to CSS variables through `@cmd-kit/core`
- slot-level class names
- renderer overrides for title, sections, items, and empty state
- programmatic navigation helpers through `useCommandPalette`
- async command sources
- recent item tracking primitives through `useCommandPalette`

Use this package when you want a working command palette quickly and still need room for customization.
